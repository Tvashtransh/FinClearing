## Authentication
All API interactions require a secure Bearer token header. Authenticate your application backend by passing your active API token inside the HTTP authorization header block. Keep your private key completely secured.

**Example Request Headers:**
```http
Authorization: Bearer YOUR_SECRET_API_KEY
Content-Type: application/json
```
